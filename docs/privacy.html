Privacidad y Términos de Silent Password

Política de Privacidad

Última actualización: Mayo 2025

Información que recopilamos

Silent Password no recolecta ni transmite ningún dato personal. Valoramos tu privacidad y hemos diseñado la extensión para funcionar completamente de manera local en tu navegador.

Almacenamiento local

Silent Password guarda tus preferencias (como la longitud de contraseña preferida, opciones de caracteres, idioma y tema) localmente en tu navegador utilizando chrome.storage.local. Esta información nunca sale de tu dispositivo y se utiliza exclusivamente para mejorar tu experiencia de usuario.

Contraseñas generadas

Todas las contraseñas se generan localmente utilizando la API de Crypto segura. Las contraseñas generadas no se almacenan ni se transmiten a ningún servidor remoto en ningún momento.

Uso del portapapeles

Cuando copias una contraseña, esta se guarda temporalmente en el portapapeles de tu sistema. Por razones de seguridad, Silent Password limpia automáticamente el portapapeles después de 60 segundos si detecta que la contraseña copiada sigue allí.

Términos de Uso

Al utilizar Silent Password, aceptas los siguientes términos:

Licencia

Silent Password se distribuye bajo la licencia MIT. Eres libre de utilizar, modificar y distribuir la extensión, siempre que se mantenga la atribución original.

Uso y limitaciones

  • La extensión está diseñada únicamente para generar contraseñas aleatorias.
  • No ofrecemos garantías sobre la seguridad absoluta de las contraseñas generadas, aunque utilizamos métodos criptográficamente seguros para su generación.
  • Silent Password no es un gestor de contraseñas y no almacena las contraseñas que generas.

Responsabilidad

El uso de Silent Password es bajo tu propia responsabilidad. No nos hacemos responsables de ninguna pérdida o daño que pueda resultar del uso de las contraseñas generadas por esta extensión.

Contacto

Si tienes preguntas o inquietudes sobre esta política de privacidad o los términos de uso, no dudes en contactarnos en contact@silentpassword.dev.
